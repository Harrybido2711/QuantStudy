## Definition

The **Sharpe Ratio** measures risk-adjusted return — how much excess return you earn per unit of risk (volatility):

$$
S = \frac{\bar{r}_p - r_f}{\sigma_p}
$$

Where:
- $\bar{r}_p$ = mean portfolio return
- $r_f$ = risk-free rate
- $\sigma_p$ = standard deviation of portfolio returns

## Annualization

To annualize from daily returns (assuming ~252 trading days):

$$
S_\text{annual} = S_\text{daily} \times \sqrt{252}
$$

```python
import numpy as np

def sharpe_ratio(returns: np.ndarray, rf: float = 0.0, periods: int = 252) -> float:
    excess = returns - rf / periods
    return (excess.mean() / excess.std()) * np.sqrt(periods)
```

## Interpretation

| Sharpe Ratio | Assessment |
|-------------|-----------|
| < 0 | Worse than risk-free |
| 0–0.5 | Subpar |
| 0.5–1.0 | Acceptable |
| 1.0–2.0 | Good |
| > 2.0 | Excellent (rare in liquid markets) |

## Limitations

- **Assumes normality**: Penalizes upside volatility equally with downside → Sortino Ratio addresses this
- **Look-back dependent**: Very sensitive to estimation window
- **Serially correlated returns**: Hedge fund returns often have autocorrelation that inflates Sharpe
- **Lo (2002)**: With monthly autocorrelation of 0.17, annualized Sharpe from monthly data is overstated by 65%
